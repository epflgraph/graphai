    Wikitext¶

    wikitext Package¶

    The wikitext package provides a simple way to interact with the concept-detection API via a python class.

    Module which provides a tool to use the concept-detection API in order to wikify a text. Wikifying a text is analyzing it and selecting a number of Wikipedia pages (concepts) which are relevant to it, together with several scores that measure this relevance.

    List of classes and functions:

    wikitext.main.equivalent(result1, result2)¶

    Returns whether two wikify results are equivalent, i.e. whether they have the same keywords and page id.

    Parameters

    • result1 (dict) – A wikify result.

    • result2 (dict) – Another wikify result.

    Returns

    True if both results are equivalent, False otherwise.

    Return type

    bool

    Returns all results for all wikitexts with the scores aggregated according to f.

    Parameters

    • wikitexts (list[Wikitext]) – A list of Wikitext objects.

    • f (Callable[list[Number], Number]) – A function to aggregate the scores of the common pages. Default: None (numpy.mean)

    Returns
