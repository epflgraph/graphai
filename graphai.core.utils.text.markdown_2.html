graphai.core.utils.text.markdown_2.parse_template(node)

Parse template node.

Parameters:

node (mwparserfromhell.nodes.Node) – Node to parse.

Returns:

Dictionary with keys ‘html’, ‘text’ and ‘aux’, containing the parsed text as html, plain text and

auxiliary text, respectively, as lists of strings. Auxiliary text are hatnotes, captions, tables and html elements under the class ‘searchaux’, and they are excluded from the plain text.

Return type:

dict
