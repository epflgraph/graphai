    Interfaces¶

    The interfaces package contains several classes whose purpose is to transparently communicate with different external services, like APIs or databases.

    interfaces.api Module¶

    Communicates with the Graph AI API.

    class interfaces.api.Api¶

    Base class to communicate with the Graph AI API.

    keywords(raw_text)¶

    Performs an API request to the /keywords entry point with the given text.

    Parameters

    raw_text (str) – Text to be used as input for the request.

    Returns

    List of keywords extracted from the given text from the response.

    Return type

    list[str]

    wikify(raw_text, anchor_page_ids=None, method=None)¶

    Performs an API request to the /wikify entry point with the given text.

    Parameters

    • raw_text (str) – Text to be used as input for the request.

    • anchor_page_ids (list[int]) – List of anchor page ids to be used as input for the request.

    List of wikify results from the response.

    list[dict[str]]

    query_wikified_slides(slide_ids)¶

    Retrieve the results of wikifying the given slides, as stored in the database.

    Parameters

    slide_ids (list[str]) – List of slide ids.

    Returns

    List of wikify results for keywords from the texts of the given slides. Each element has the following keys:

    • ’keywords’ (str): Set of keywords.

    • ’page_id’ (int): Id of the wikipage.

    • ’page_title’ (str): Title of the wikipage.

    • ’searchrank’ (int): Position of the wikipage in the wikisearch.

    • ’median_graph_score’ (float): Median of the graph_score over all anchor pages.

    • ’searchrank_graph_ratio’ (float): Ratio graph_score/search_score.

    Return type

    list[dict[str]]

    get_wikipage_ids(filter_orphan=False)¶

    Retrieve a full list of all wikipage ids present in the database.

    Parameters

    filter_orphan (bool) – Whether to filter out wikipages with no links from or to other wikipages.

    Returns

    List of wikipage ids.

    Return type

    list[int]

    get_wikipages(ids=None, id_min_max=None, limit=None)¶

    Retrieve wikipages and related information from the database.

    Parameters

    • ids (list[int]) – If set, restrict to wikipages with id in this list.

    • id_min_max (tuple[int, int]) – If set, restrict to wikipages with id in the given range.

    • limit – If set, limit the number of wikipages to this number.

    Returns

    Dictionary with wikipage ids as keys and whose values contain the following keys:

    • ’title’ (str): Title of the wikipage.

    • ’content’ (str): Content of the wikipage.

    • ’redirect’ (list[str]): List of titles of wikipages redirecting to this one.

    • ’popularity’ (float): Popularity score of the wikipage.

    Return type

    dict[int, dict[str]]

    get_wikipage_categories(ids=None, id_min_max=None, limit=None)¶

    Retrieve wikipage categories from the database.

    Parameters

    • ids (list[int]) – If set, restrict to wikipages with id in this list.

    • id_min_max (tuple[int, int]) – If set, restrict to wikipages with id in the given range.

    • limit – If set, limit the number of wikipages to this number.

    Returns

    Dictionary with wikipage ids as keys and a concatenation of the category titles as values.

    Return type

    dict[int, str]
